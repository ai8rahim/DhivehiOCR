Hardware Requirements

Recommended hardware specification

  • Personal Computer with Intel® Pentium® 4, 2.0 GHz Processor
  • 256 MB RAM DDR
  • 20 MB Free Hard Disk space for system installation
  • 64 MB Graphics Card
  • Scanner that support 300dpi resolution scanning.

 

Software Requirements

The following software has to be installed in order to run the system.

  • Microsoft Windows XP Pro/Home edition
  • Java TM 2 Platform Runtime Environment 5.0
  • Microsoft Internet Information Service (for web server)
  • MySQL database (for web server)

 

 

Installation Instructions

To install the application, just copy the EPublisher directory from the installation CD to the system. The rest of the required libraries are bundled in with the system.

 

Server Side Installation

First the database has to be setup. To do this, you should first create a database with the name epublisher, and create the user root with password root.

 

Once this is done, you could create the tables by using the SQL scripts in the sql directory in the installation directory.

 

The next step should be to copy the epublisher directory and the index.html and applet.html file to the desired location in the InetPub directory of the web server.

 

Running the System

To run the system, you could simply run the batch file located in the installation directory called epublisher.bat.
